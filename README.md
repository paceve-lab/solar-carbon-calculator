# 🌞 Solar Carbon & Savings Calculator

Calculator for **Electricity Bill Savings + Carbon Credits** for homes and small & medium businesses in Brazil.

### 🎯 What does this tool do?
- Estimates **annual savings** on your electricity bill with a solar photovoltaic system
- Calculates the **potential carbon credits** generated
- Uses average solar irradiance by region in Brazil
- Based on updated data from MCTI, ANEEL, and typical tariffs

### 📁 Project files
- `solar_carbon_calculator.py` → Main Streamlit application code
- `requirements.txt` → Python dependencies
- `index.html` → Static page that embeds the calculator (used by GitHub Pages)

### ⚠️ Important notes
- The values are **estimates**. Actual results depend on your electricity distributor, contracted tariff, roof orientation, and shading.
- Carbon credits for small systems usually require project aggregation and certification.
